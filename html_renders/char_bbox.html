Label Characters from the Flintstones

Please use your mouse to draw a box around every character in this image. You should start with any and all of the main characters shown below. For these 10 main characters, a menu will appear after you've drawn a box, and you should select the correct name to label that character. If there are additional characters not in the main 10, press the "Label Other Characters" button, and a new image will pop up below to work on. After drawing boxes around one of these characters, a box will appear for you to enter a short description. For example- "man with bow tie", "construction worker", or "circus performer". If there are no characters present, draw a box anywhere and select the label NO CHARACTERS.
